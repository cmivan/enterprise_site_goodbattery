+  医院简介 >>  网站制作之SEO技巧 +
　　网站制作之SEO技巧：
　　1. 首先是Meta 标签可能这个已经众所周知或你已经在使用，但是如何才是有效的使用呢？ 在此我们说两点。 枢纽词凸起性是指枢纽词间隔页面起始处所隔的间隔，枢纽词权重是指某个特定的枢纽词或句子在页面上面所泛起频率，枢纽词密度是某个枢纽词在整个页面字符中所占的比率， 权重和密度都不宜太高，这样会被引擎以为有枢纽词堆砌之嫌，将可能受到搜索引擎的一些惩罚。 （2） 免费的。
　　2. 关于枢纽词需要说明的一些要点搜索引擎在决定一个站点排名的时候会评估枢纽词的凸起性，权重，枢纽词密度。 这也是另外一种放置更多的枢纽词到HTML当中，改善页面枢纽词权重，密度的方法。心提示：最难的地方就是使你的枢纽词获得最靠前的排名，下面将讲述一些简朴的有效的对搜索引擎有效的方法来晋升站点排名和网站流量。 同样，也不要在这些标签下使用 “和” 或者”或” 之类的词。 一个简朴的alt标签如下： alt=“put your keyword phrase here.” ，搜索引擎会也会单独的评估alt标签中的枢纽词的凸起性，密度，权重，所以标签的优化也势在必行。 Alt标签使用起来都比较简朴，它们能够使你的站点枢纽词排名起到大为不同的作用。 将导航链接（以及Java脚本）放在右边或者底部，而不要放到页面的左边。 搜索引擎将会非常正视站点的前100词，你总不但愿那个这些重要位置的词是导航链接或Java脚本吧，应该做的是将抬头标签及枢纽词放到页面起始处。 使搜索引擎知道图片内容的独一方式就是通过alt标签。一是title标签，另外就是描述标签，而枢纽词标签我们不做说明，由于现在搜索引擎已经不太正视这点，甚至有些争论以为它根本就没有任何作用，但是我们依然在使用它，由于它还有有其一定的长处的，并且使用此标签也没有什么坏处。 我们发现在”标题”和”描述” 这两个标签下使用相似的文本并将枢纽词放到这两个标签显眼的位置是很有效的（位于开头处并且多于一次），我们发现良多的站点在其title标签中使用 sitename.com， new page1，或者 welcome to my site 等，这些事实上起不到晋升某些特定枢纽词排名的作用。
　　3. 在所有的图片上放置alt标签对搜索引擎有一定了解的都知道搜索引擎是无法抓取任何的图片或者影像的。 搜 索引擎读取页面的时，它的顺序是从页面的左上端到右真个底部。
　　绝大多数的网站和贸易站点所寻求的一个共同点都是站点流量，每个网管都知道下最好的流量类型是天然的有机的搜索引擎流量，有两个重要的原因： （1） 目标点很明确。以上的这些对于单个的页面，标题标签，标签描述都将单独的进行评估。 或者说，将导航链接或者Java脚本放到页面的右端或者底部，这样可以让搜索引擎可以更好的抓取文本，这样才将重要的位置做到了‘位’尽其用。
　　以上就是网站制作之SEO技巧，希望能帮助大家！
