+  医院简介 >>  网站策划概要 +
　　网站制作不像良多人想得那样,直接可以写代码或程序搞好就行的.在做网站时.最重要的是要做好策划.
　　制作一个网站都要做哪些基础策划呢?
　　1.网站类型及服务策划这是制作网站最先要考虑的事情.由于这一步没有决定好.后面的工作无法开展.
　　考虑一个网站的类型与服务当然,对于个人网站来说,内容不用考虑太全面.但一定要专.这样以后比较好治理与发展.
　　2类型与服务已经定下来后,就是要选一套程序像上面说的这样的文章站,完全可以选择PHPCMS等内容治理系统进行建站.假如要建一个论坛或交流的网站,就可以选择像Discuz等程序.
　　当然,假如你有技术或技术团队.又没有找着适合自己的程序,可以自己开发一个程序.如:网页制作教程网就是自己开发的程序3选择了一套程序后,要根据自己网站提供的内容或服务来制定网站色调或分格等.这个期段包括网站LOGO的设计.模板的选择等工作.
　　4.最后就是要制定一个推广方案及内容建设方案.
