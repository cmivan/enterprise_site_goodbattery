+  医院简介 >>  网站大小与网站流量 +
一、网页大小这里的网页大小是指网页的字节大小，网页的物理大小。
很多搜索引擎的搜索结果中会看到网页大小。
我们知道，在内容和功能确定的前提下，网页字节数越小，打开速度越快，对se爬行越有利，这个网页大小具体指的是么呢?它指的是该网页源代码的字节大小，不包含额外的js，gif，flash，jpg等。
怎么得到网页大小的值呢?

二、网页流量很多主机都有流量限制，这里说的流量就是指从服务器端下载的实际流量，即用户浏览网页产生的实际流量。
1、用户第一次打开网页A时，会将网页A本身(网页源代码的字节大小)+Css+Js+网页中的图片+flash+其它全部下载到本机硬盘(浏览器缓存)中，这是的流量等于以上文件字节数的总和;2、当用户再打开网页A之后，又打开了网页B，而网页B中包含了部分和A相同的CSS，JS，图片，同时又包含自身的一部分图片，js等，这时浏览器不会下载这些相同的文件，所以这是打开网页B的流量=B源代码+网页B中新的图片，cs，js等;3、当用户再打开网页A、B后(没有删除缓存)，用F5刷新了A，或B，浏览器只会重新下载A、B源代码;4、当用户用使用Ctrl+F5刷新A、B，或清除了缓存，重新下载A、B，同前两步;F5为读取缓存，Ctrl+F5为重新从服务器读取。
