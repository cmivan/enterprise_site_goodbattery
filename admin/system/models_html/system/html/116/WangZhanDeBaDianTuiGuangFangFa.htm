+  医院简介 >>  网站的八点推广方法 +
　　第一：就是要加入网络上有的各大引擎，像什么百度，google，雅虎，搜狗，Alexa等只要有的都加入，像一些不知名的也要加入，如快搜，天网搜等等，不要小看这些小的搜索，时间长了带来的流量是不少地，细水长流嘛。我网站里有各大引擎收录的地址.
　　第二：把搜索联盟代码放到你的首页，看你想放什么联盟的，一般不是百度就是google，但是有一点一定要记住，一个网站内不能放两个联盟的搜索代码，你放百度的，网页被百度收录的就快，反之google也是，这就要看你喜欢那个了。
　　第三：加qq群网站推广法。用你所有的qq加群，能加多少加多少，每个群每天发一次你的网站地址和介绍。如果你的qq够多，加的群是大群的话，宣传的效果就不用提了，流量呼呼就来了。不要怕被踢，qq群多的是，这法好像有点那个，但是为了我们新网站要忍辱负重脸皮要厚。
　　第四：给其他网站提供免费的服务，收集一些网站不容易整到东西，根据你的网站自己结合，比如免费统计、免费素材、免费电子书等等，时间长了，流量慢慢就来了，这是一个吸引人方法，现在不免费谁看啊。
　　第五：电驴、bt、哇嘎推广方法，把你的网址放到音频和视频里，借助下载软件下到千家万户，要放到最新，最流行的音像文件里，比如最新的电影，最新的专辑等，这需要一定的技术含量，整的好的话，一个视频文件，就能来几万ip，自己想吧。
　　第六：论坛，浏览本，贴吧等聚集网民的地方，那热去哪，发表热门话题，自己顶自己，关键要有一个好的标题，标题可以写暧昧的话题，注意不要发违法的东东，警察叔叔来找你了可不愿我啊，百度贴吧不用说大家都知道，那可是个好地方，在百度贴吧里宣传你的网站时，一定要有足够的马甲，还要经常换ip，不然的话，你自己试吧!
　　第七：加链接，不用多说了，大家都知道，我认为不用加那么快，一两个星期加一个就好。
　　第八：网摘推广，这可是网站推广的非常好的办法，只要你写的文章好推荐的好，你会发现从网摘过来的人是惊人的，下面是大的网摘。新浪VIVI，365Key天天网摘，POCO 网摘，CSDN网摘，等自己搜吧，还等什么，赶快去投去吧。推荐文章时一定要注意，精，还是精，文章不在多，在于精，还要仔细阅读下他们的说明，倒是因为xx封你就不好看了。
　　最后说一点，不要怕麻烦，不要怕付出没有收获，还有就是不要有太大的幻想。很多方法不是一推出就可以出效果的，而是一种长期付出，长期量的积累，只要你付出了，就一定会有收获，现在网站多如牛毛，要想在如此多的牛毛中间显示自己，自己想吧。大家要学会举一反三，结合自己的网站有针对的总结出推广方法，最后祝大家都取得成功。
