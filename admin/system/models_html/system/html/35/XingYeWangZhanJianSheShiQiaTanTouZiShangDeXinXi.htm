行业网站建设时洽谈投资商的信息:

【洽谈投资商信息】：点击图 4 10“新增”按钮可以添加“项目洽谈签约信息”。选填相关信息点击保存即可。

【规划设计信息】：点击“新增”按钮可以添加“规划设计信息”。选填相关信息点击“保存”按钮即可。

【立项审批信息】：点击“新增”按钮可以添加“立项审批信息”。选填相关信息点击保存即可。

注意：新建在建项目点击“新增”按钮显示新增在建项目页面。如图图在建项目新增操作和前期项目基本相同。

【项目性质】：分新建、续建、改建、扩建。

【投资单位信息】：为必填项，几个投资商出资比例加起来必须为例如：投资商A 的投资比例为整个项目的投资商B 的投资比例为整个项目的修改、审核、退报在建项目【修改在建项目】：点击项目列表的项目名称可以修改选择的项目。图本来项目的属性为锁定不可修改。点击“修改”按钮可以把项目属性激活。输入修改的信息点击保存即可。

【审核在建项目】：点击项目列表的项目名称可以审核选择的项目。检查项目信息后点击“审核”按钮激活审核状态，让审核下拉按钮激活可选状态。选择好审核状态后保存即可。审核后即上报到上一级机构。例如：登录用户为 “宁波项目管理员”审核完成后“浙江项目管理员”就能看到此项目。

注意：项目审核过后将不能 “修改项目”、“删除项目”。

【退报在建项目】：与第三章取消上报操作相同。

修改在建项目投资结构点击“修改投资结构”就可以修改“投资单位信息”，修改好保存即可。

注意：只有建此项目的机构才能看到“修改投资结构”按钮。

在建项目图片管理与展示在建项目图片管理与展示是保存此项目建设期间的图片信息。

点击“图片管理与展示”按钮会显示如图图点击“新建”按钮显示填好相关信息保存即可。

图进度操作图注意：只有建此项目的机构才能看以上四个按钮【新增进度】：对项目进度的上报。只有审核过的项目才能进行进度上报操作。点击图“新增”按钮默认为本期的进度上报。填好相关信息保存即可。
