企业网站建设中队线的测试：

当然，对网线进行专业的测试，需要比较昂贵的设备。我们若为测试网线而特意去购买这些设备的话，那也太小题大做了。一台测试网线的设备，不知道可以买多少网线了。笔者这里介绍几种价廉物美的方法，供大家参考。

一、 网线能否久经温度的考验在组建企业网络时，有时候网线都布在密不透风的产所，而且，有时候又难免要受到太阳的照射。所以，我们在选择网线时，对于刚开始网络组建时，一定要考虑网线性能对温度的敏感程度。如在冬天时，网线能否在零度以下的温度中正常工作。同时，还要考虑当温度比较高时，如在40度左右的温度，网线会否发生一些物理性的变化，如表皮会不会变软，等等。

网线信号跟温度有很大的联系。根据专业部门测试，温度每升高十个摄氏度，网线的信号衰减就增大4%。这意味着40摄氏度下92.6米线缆的信号衰减与20摄氏度下 100米线缆的信号衰减相同。所以，温度对于信号衰减的影响及作用要远远大于许多其它的固定因素。而根据有关部门测试，在夏天，一些混泥土结构层中的网线温度高达40多度。在一些企业中，可能会更高。

可见，温度对于网线的影响，是非常巨大的。

简单测试我们在组建企业网络的时候，需要对我们所买的网线进行温度的测试。可能我们没有条件测试低温情况下的网线性能。但是，很明显，我们可以对高温情况下，网线的一些性能进行测试。对于温度来说，我们主要测试的是，在高温情况下，网线的外表皮会否明显的软化等等。

按照相关标准，网线外表皮应该能够忍受50度的温度不变形、不软化才算满足基本要求。不然的话，若不能满足这个要求，那么在企业中的应用寿命不会太长。

我们可以截取一小段网线，在火炉旁边烤;或者直接拿到太阳底下晒。若在50度左右的温度下，若网线能够坚持20分钟左右没有发生软化现象的话，那这网线就过关了。

秦皇岛网奇公司从事多年企业网站建设多年,专业技术值得信赖。
