网站制作需要关心的问题----门户网站制作

拿一跟一米长的网线，来回的弯。至少要来个二是个回合。然后在适当的拉一下，力度稍微大一下也没有管理。然后给网线两端装上水晶头，看看该网线是否可以用。若不可以用的话，那一般这网线就不符合标准，除非是你拉的时候，太用力了。

把网线批剥出来，看看里面铜线的亮度如何。一般来说，若网线是采用回收的废铜制作，或者采用的铜含有比较多的杂质的话，网线的亮度都不会很亮，而是带有一点灰色，就好象铜被氧化的那种现象。若是正牌网线的话，其光泽会很亮，而且是六跟铜丝，每根都那么的亮。不会有一根特别的暗一点。

正规的网线的外表皮还具有一定 延展性，或者叫做弹性。为什么这么说呢?想想我们在电线管中穿线的话，若同时穿的线多的话，空间就会比较小。若外表皮没有弹性的话，就很容易断裂。所以，我们在购买网线的时候，可以减一段0.5米长的网线，向两边拉。看看中间有没有裂缝，或者表皮能不能向两头延伸。若在拉的过程中，中间没有裂缝，而且，表皮也可能向两头延伸的话，则说明这个网线的表皮符合标准的。另外要注意一个问题，在拉的过程中，有的网线虽然没有裂缝，但是，在中间有明显表白、表薄的迹象，这也是不符合标准的。

利用游标卡尺量网线铜丝的直径。我有时候去买工程用的线缆时，会随身带着一把游标卡尺。有些网线其表面功夫做的非常的好，如表面的印字、商标等等都跟正牌网线一模一样，看不出有什么区别。但是，利用游标卡尺一量的话，就会发现问题的所在了。正规的网线，其内部一根铜丝的直径一般在0.5毫米左右;但是，杂牌的或者冒牌的网线，一般直径都只有0.3毫米，最多只有0.5个毫米。这么微笑的差别，利用肉眼是看不出来的，我们只有通过仪器，如游标卡尺进行测量才能得知。
