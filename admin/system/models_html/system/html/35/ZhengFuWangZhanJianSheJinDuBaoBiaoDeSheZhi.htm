政府网站建设进度报表的设置:

这里的项目进度上报的时间和项目里的进度上报方式一致如图图每一个阶段只能上报一次【补报】：对漏报 阶段的进度进行补报【修改计划】：对“项目进度信息”里的“计划投资”和“建设内容”的修改。点击“修改计划 ”按钮然后修改“计划投资”或“建设内容”点击保存即可。

【竣工验收】：把竣工的项目设为竣工状态。点击“竣工验收”按钮，填写“项目进度信息”里的“验收情况”点击 保存即可。

注意：设为竣工状态后不能对此项目有如何的修改。

【项目进度信息的修改】如图 4 28点击“修改”按钮在弹出的页面修改相关内容后点“保存”即可。

图【项目进度信息的审核】如图4-28点击“进度审核”按钮在弹出的进度审核页面进度审核选择已审核后点“保存” 即可。

【项目进度信息的删除】如图4-28点击“删除”按钮即可。

审核上报点击菜单栏中的“项目管理”再点击其子栏目“审核上报”会显示如图图这里显示了前期项目和在建项目没 有审核过的项目，可以在这里批量审核，审核过后就会上报到上一级机构。上级用户可以查看到下级上报的项目。

【审核上报项目】：在项目前打“√”点击后面的“审核图标”按钮即可。

综合查询点击菜单栏中的“项目管理”再点击其子栏目“综合查询”显示如图图可以设置不同的查询条件对“前期项 目”、“在建项目”和“竣工项目”进行查询。

统计分析统计分析模块主要用于项目信息的统计、上报、分析等。

点击菜单栏中的“统计分析”显示如图图报表产生各级根据系统实有信息数据，直接产生各种报表。
