+  医院简介 >>  网站优化基本概念 +

　　所谓网站优化，指迎合搜索引擎的网站排名规则进行网站的规划，网页的设计，以确保网站能在搜索引擎上获得较好的排名，让更多的潜在客户能够很快地找到从而求得网络营销效果的最大化。

　　一、寻找关键词技巧

　　单靠自己的努力有时难免会有些遗漏，你总是能找到最能反映自身业务特点的关键词。这时你无妨来到搜索引擎上，找到竞争对手的网站，看看他使用的哪些关键词，也许会从中得到一些启发的

　　借助一些关键词自动分析软件，此外。可以迅速地从你或你竞争对手的网页中提取适合的关键词，使你工作效率成倍地提高，可以借助这些软件找到许多以前不曾考虑到关键词，从而大幅扩充我关键词列表。

　　二、停用词/过滤词

　　都是指一些太常用以至没有任何检索价值的单词，这两者意义一样。比方“atheandofwebhomepage等等。搜索引擎碰到这些词时一般都会过滤掉。因此为节省空间，应尽量防止使用这一类的词，尤其是对文字数量有严格限制的地方。如想验证上述规则，可以试着在搜索引擎中以“staithenight查询。会发现结果中单词“the虽与搜索条件匹配，但并未以粗体字显示，说明它被忽略掉了

　　三、重复关键词

　　既然关键词出现频次是决定网站排名的重要因素。为何不将它多次重复呢，这样不是即简单又有效吗？殊不知这可是搜索引擎“石器时代”做法，当时的确奏效，现在则早已被搜索引擎所摒弃。

　　关键词......重复次数越多，网站排位越靠前。于是登录搜索引擎变成了一场无休止的关键词重复大赛。重复500次不算多，来600次，而将关键词重复上千次的也不乏其人。搜索引擎很快意识到这种做法的危害性，忍无可忍之际终于站出来，警告那些事情做过了头的网站，如果继续执迷不悟，不只达不到目的还有可能被处极刑―永远将你网站拒之门外！尤其不要在同一行连续使用某个关键词2次以上。使用关键词时，不要刻意过分重复某个关键词。要尽量做到自然流畅，符合基本的文法规则。

　　三、关键词的使用

　　也许还记得搜索引擎是如何对用户的查询作出反应的首先以输入的关键词为搜索条件，寻找关键词的目的就是为了要将它用在网页中。看了前面有关搜索引擎工作原理以及网页排名规则的介绍。其数据库中检索包括该关键词的网页，然后依照“匹配/位置/频次”原则返回网站排名搜索结果。

　　必须在网页中使用关键词，放置关键词的地方包括：标题、META 标签、网页正文、ALT标签等等。但有一点要注意，  所以为了让你网页呈现在以事先确定的关键词为条件的搜索结果中。不要过分重复某单个的关键词，而且也不能象在META 关键词标签中那样简单地排列关键词，而是应该遵循必要的语法规则，形成自然流畅的语句，让访问者看起来觉得很舒服，同时还要顾及对顾客是否具有足够的吸引力。总之牢记一个原则：对访问者有吸引力和价值的内容，搜索引擎也不会熟视无睹。
