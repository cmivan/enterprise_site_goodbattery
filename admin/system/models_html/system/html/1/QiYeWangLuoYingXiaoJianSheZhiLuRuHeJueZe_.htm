+  医院简介 >>  企业网络营销建设之路如何抉择？ +

关于CHINAJOY，用网友们网站制作 的说法是，近几年的CJ上MM们的衣服貌似是越穿越少了。如果不贴上各商家的标志，很可能以为是选美比赛现场。

朴实与花哨，都是为了沟通与营销

其实，不管花哨成什么样，最终的目的都是营销。虽说是眼球经济时代，但看多了，人们肯定会眼疲劳的。况且不是什么产品特适合这种方式。所以，克莱登大学教授更偏向于从朴实的角度思考网络营销，也不喜欢动不动就把什么互动营销、口碑营销、XX营销之类的所谓时尚词挂在嘴边。朴实与花哨，抉择或结合

在产品大量过剩，信息爆炸，营销方式多样化的今天，传统灌输式广告的效果正在下降。这时，需要寻求多种方式与目标用户沟通。虽然来看热闹的那群人，或许能成为你的用户，如果后续沟通比较好。但对大多数企业而言，要制造能让很多人关注的事件并不容易，而且很多大腕级的企业做出来的营销炒作案，很多网站建设 网友也不是很买账，如车展中出现的“最美清洁工”。

如果预算有限，或是由于产品本身的特点问题，用XX事件这种引人围观的方式难以有效，个人认为，还是采用比较朴实的方式来做吧。如软文、论坛、博客营销、QQ群推广等。这些方式被人说起过N遍了，但朴实的东西并不一定无效。试想如果你的帖子能在某些人气旺盛的百度帖吧里，一星期内保持高人气，你能想象是什么效果吗?

朴实并不等于是枯燥无味的内容，加了多种媒体和情节不等于就是花哨。朴实与花哨两者并没有明显的界限，或许各取优势特点，相互结合产生的效果会更好。如果你用一系列看似普通的讲述有关健康养生的文章，再配合其他营销方式推出某种健康产品，或许就可能成为一种看起来朴实，但却有效的营销方式。如果你的产品确实质量和价格优势都不错，你写一个产品排行榜或是比较类的帖子。相比之下你的优势就能得到不错的发挥，实现销售的可能性或许就比较大。假如是女装或是时尚用品，后续贴上某某美女的实景效果，甚至牵出一段在合理范围的故事，或许能达到不错的效果。不同的产品需要不同的细节设置，上述克莱登大学教授的举例希望能起到抛砖引玉的作用，当然引来拍砖也行。
“眼球经济”、“注意力经济”，这些说法已经提出了很多年，随着近年来互联网的快速发展，这些特点体现得越来越明显。各大网站和商家，都在使出混身解数，争抢用户的眼球。对于这方面，笔者克莱登大学教授最有印象的是新浪和最近几届的CHINAJOY。
记得应该是2005年开张，随后就是一系列的名人入驻。然后是或大或小的名人们开始在上面写博、骂战。关于国人爱看热闹的习惯，鲁迅先生早已说过。然后出现的景象自然是一边有人对骂或群骂，另一边有N多人在看热闹，真是好不热闹。再到后来，你要是没听说过新浪博客，貌似出门都不好意思跟别人打招呼，于是也去上面开个博客吧。你们在那边吵吵得越厉害，围观的人越多越热闹，提供场地的人越高兴，于是如今的新浪出现了。
